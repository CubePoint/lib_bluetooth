# 目标蓝牙连接库 (功能开发中。。。)


## 项目描述

* 主要应用场景：对有明确蓝牙设备目标进行连接。
* 对不同平台蓝牙连接API进行了统一封装，便于代码移植。
* ...


## 项目介绍

* 主要类 Bt BtService BtBaseService
  * Bt：统一操作类
  * BtService：各平台API实现类
  * BtBaseService：基础实现类
* ...
